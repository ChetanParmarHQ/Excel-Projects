# Excel-Projects

Excel-based analytics projects featuring dashboards, KPI tracking, financial models, data cleaning, reporting, and business insights. This repository is a growing collection of hands-on projects that demonstrate practical Excel and data-analysis skills.

---

## 📌 About This Repository

This repo brings together end-to-end Excel projects, each with its own workbook, documentation, and (where relevant) a presentation. Every project focuses on turning raw data into clear, decision-ready insights using formulas, pivot tables, charts, and financial models.

**Skills demonstrated across projects:**
- Data cleaning and preparation
- Formulas and functions (lookup, logical, financial, statistical)
- Pivot tables and dashboards
- KPI tracking and reporting
- Financial modeling and analysis
- Data visualization with charts
- Business insights and recommendations

## 🗂️ Repository Structure

```
Excel-Projects/
├─ Proect-1-Depreciation-Calculator/
│  ├─ Project-1(Depreciation-Calculator-Excel).xlsx
│  ├─ Comparative Analysis of Depreciation Methods.pdf
│  └─ Project-1(Depreciation Calculator Project Documentations).pdf
└─ README.md
```

Each project folder typically contains:
- The **Excel workbook** (`.xlsx`) with all calculations and charts
- A **documentation** file describing the scope, method, and findings
- An optional **presentation** summarizing the analysis

---

## 🚀 How to Use

1. Browse the **Projects** table above and open the folder you are interested in.
2. Download the `.xlsx` workbook to explore the formulas, schedules, and charts.
3. Read the accompanying documentation/presentation for context and conclusions.
