# Excel Assignments

A collection of six Excel assignments completed as part of the CoachX training program. Each workbook focuses on a specific Excel skill, from logical and lookup functions to business and financial analysis calculators.

Maintained by **Chetan Parmar**.

---

## About this directory

This folder groups practical Excel exercises that build core spreadsheet skills used in data entry, MIS, and analyst work. Every assignment takes a real world style scenario and solves it using Excel formulas, functions, and structured layouts.

## Assignment details

### 1. Logical Function
Uses an employee dataset to decide an action (for example, terminate or reward) based on a performance condition such as Very Poor to Excellent. Practises IF and nested logical functions.

### 2. VLookup Functions
Links a large sales order sheet (over 4,800 rows) to a customer information sheet (500 customers) using VLOOKUP, pulling customer details like segment, city, region, and age into the orders.

### 3. Online Sales Tracking Analysis
An online sales tracker with a sales summary and report sheet. Builds a structured tracker and derives summary figures from the raw sales data.

### 4. Income and Business Analysis
Contains a horizontal analysis of an income statement (year over year variance) and a business net worth calculator that nets assets against liabilities.

### 5. Startup Costs Calculator
A startup budgeting template comparing proposed against actual amounts across investment and expense categories, with a difference and summary section.

### 6. Overtime Percentage Calculator
Calculates each employee's overtime as a percentage, turning raw hours into a clear overtime metric.

## How to use

1. Download the assignment workbook you want to review.
2. Open it in Microsoft Excel, Google Sheets, or LibreOffice Calc.
3. Check the problem statement or header section for the task.
4. Click cells with results to see the underlying formulas and functions.
